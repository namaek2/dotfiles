# Hyprland Dotfiles

Mixed JaKooLit's and 1amSimp1e's one.

# OverView


# Dependencies

-   OS : Arch Linux
-   WM : Hyprland
-   Device : XPS 13 9300 4K
    -   if you are not on 4K, might need additional configuration
-   Bar : Waybar
-   Launcher : Rofi
-   Logout menu : Wlogout
-   Notification : dunst
-   Terminal : kitty
-   Shell : Oh My Zsh
-   Fetch : neofetch
-   Fonts :
    -   font Awesome
    -   JetBrains Mono Nerd
-   lock : hyprlock
-   idle : hypridle
-   icon : papirus theme

# Installation

```terminal
paru -S hyprland-git swaylock waybar-hyprland-git kitty-git oh-my-zsh-git neofetch-git rofi-git wlogout dunst
```
